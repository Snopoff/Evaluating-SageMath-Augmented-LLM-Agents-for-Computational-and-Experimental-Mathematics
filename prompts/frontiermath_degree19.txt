------------------------------------------------

SAGE EXECUTION RULES

Code is executed using

    sage -python

This behaves like standard Python with Sage libraries.

Important consequences:

• exponentiation uses `**`
• `^` is NOT exponentiation
• Sage notebook syntax such as

    R.<x,y> = PolynomialRing(QQ)

  is NOT supported.

Instead use:

    R = PolynomialRing(QQ, ['x','y'])
    x, y = R.gens()

Additional rules:

• Sage symbols are already available from `sage.all`.
• Do NOT import `sageall`.
• Do NOT use external CAS libraries such as SymPy.
• Always assign the final computed value to a variable named `RESULT`.

Example:

R = PolynomialRing(QQ, ['x','y'])
x, y = R.gens()

f = x**3 - y**3
fac = f.factor()

RESULT = fac
