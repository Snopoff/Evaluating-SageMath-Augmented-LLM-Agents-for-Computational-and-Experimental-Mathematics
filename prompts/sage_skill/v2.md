# General Rules of SageMath Usage

- Use I.reduce(f) for ideal remainders.
- Avoid full dir(obj); use targeted hasattr checks.

## Univariate polynomial coefficients

- Do not use p.coefficient(x) or p.coefficient(x^k).
- Use p[k] for coefficient of x^k.
- Use p.monomial_coefficient(x^k) when you want monomial-based access.
- Example:

```sage
  R.<x> = PolynomialRing(QQ)
  p = x^19 - 19*x
  linear = p[1]
  even_terms_zero = all(p[k] == 0 for k in range(0, p.degree() + 1, 2))
```
